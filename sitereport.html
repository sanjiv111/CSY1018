1. Executive Summary

Portfolio website is generally a extension of resume. It provides the way of potential clients to view his/her work, skills and educational background. There are a lot of skilled and highly qualified candidates but unable to pick up them. so, this site system helps to connect and review his/her profile for the suitable designation. so, The assignment is based on the portfolio web development. HTML and CSS are the primary code to design and implement it. There are 5 sections: index.html, portfolio.html, sitereport.html, cv.html and contact.html. The site is responsive in mobile freindly environment, desktop and various browsers. Each page use the same website and can be reached using the navigation menu. Hamburger menu system is used in mobile version. Most of the fonts, colors, UI design in this site were inspired from some of the famous celebriites like: Noel Gallagher, David Attenbourgh, Ronaldo, etc.

8. GIT HUB

Github is the internet hosting for software development and version control using git. We can work as team or individual and update our work regulary. Some of the github command that i have learnt in this term are given below:

GIT STATUS

  • git status - this command is used to check the status of the repository files in your desktop.

GIT ADD

  • git add "filename" - this command is used to add the files in your github through cmd or gitbash.
  • git commit -m "comment"- then in this command we commit or make comments for the files added.

GIT PUSH

  • git push - this command is used to push your files to github project online.
